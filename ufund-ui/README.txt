browse-needs component
    The browse needs componet will create a list of all the available needs that the user may checkout with.
    The needs will be displayed with their name a description and a cost.
    A button will be present to add the need to the user's checkout basket.
    Clicking on the need's name will allow you to view a detailed page on the specific need.
